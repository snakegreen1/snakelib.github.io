깃블로그 만들기

  1. 1단계
    • ???.github.io 레포지토리 생성 (pubilc, README 체크)
    • Setting - Pages - Source - 'Deploy from a branch' 설정
    • 사이트 접속 확인하기 (https://>>>.github.io)
    • git clone 후 index.html 파일 생성, 커밋, 푸시
  2. 2단계
    • Ruby 설치, 'Start Command Prompt with Ruby'에서 명령어 입력
    • 이동:
      cd C:\Users\karina\Documents\karina.github.io
    • 설치:
      gem install jekyll bundler
      gem install webrick
    • 설치 확인:
      ruby -v
      jekyll -v
      bundler -v
    • Jekyll 서버 구축:
      jekyll new ./
      # 또는
      jekyll new ./ --force
    • bundle 설치:
      bundle install
    • 서버 실행:
      bundle exec jekyll serve
    • 포트 접속 확인
  3. Jekyll 테마 적용
    • 테마 선택:
      http://jekyllthemes.org
      https://jekyllthemes.io/free
      https://themes.jekyllrc.org
      https://github.com/topics/jekyll-theme
    • 테마 적용:
      https://github.com/cotes2020/jekyll-theme-chirpy
      모든 파일, 폴더를 로컬 레포지토리에 복붙
    • 루비 프롬프트:
      bundle install
      bundle exec jekyll serve
      서버 실행
    • 깃허브:
      Setting - Pages - Source - Github Actions 설정
      Configure 클릭
      Commit changes... 클릭, 로컬에서 pull
    • Node.js 설치
      https://nodejs.org/en/
      루비 프롬프트:
      npm install && npm run build
  4. 테마 상세설정
    • .gitignore 하단 수정:
      # Misc
      # _sass/dist
      # assets/js/dist
    • _config.yml 수정:
      timezone: Asia/Seoul

      url: "https://karina.github.io"

      github:
      username: karina
  5. 사이트 반영 확인
